LzFunc - Less Boilerplate

This is a documentation to a Python module called "LzFunc". The module includes some functions that can simplify the code and reduce the "boilerplate", so the programmer doesn't have to grind down his fingers to write 5 more characters.

There is a full documentation for all the functions LzFunc that includes:

Downloading LzFunc

The process is very simple: You just have to enter this line in the Console with PIP installed:

pip install LzFunc

And then open your .py file and type in this line that will import all of the functions from the module:

from LzFunc import *

By doing these steps, you'll be able to use all LzFunctions in your algorithm. By the way, you can check out a built-in documentation by typing this line:

help(core)
nl()

Skips a line in the output. Good for utilities and text games.

print("Hello")
nl(2)
print("World")
> Hello
>
>
> World
pause()

Waits an interval with a three-dot animation. Can also be used in some utilities in a fake download or as a visual effect.

print("Hello")
pause(3)
print("World")
> Hello
input_yesno()

Makes simple "Yes or No" question, that returns complete boolean answer to the variable. (Has different yes or no answers: yes, y, ofc, of course, duh, yeah, yea, no, n, nah, naw, nuh uh).

answer = input_yesno("Do you like dancing? ")
if answer == True:
    print("I like trains.")
if answer == False:
    print("I like singing!")
> Do you like dancing? Yes
> I like trains.
input_default()

Returns Default value on [ENTER].

name = input_default("What is your name? (Anonym by Default) ", "Anonym")
print(name)
> What is your name? (Anonym by Default) [ENTER]
> Anonym
invert()

Simple switch to invert the variable.
* No way it's simple, I spent half an hour making this function.

switchstate = True

if switchstate == True:
    turn = input_yesno("Turn off the switch? ")
if turn == False:
    None
if turn == True:
    switchstate = invert(switchstate)
print("Current switch state:", switchstate)
> Turn off the switch? Yes
> Current switch state: False
